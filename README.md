# 98堂-115离线助手

一个 Violentmonkey 用户脚本，用于从98堂论坛帖子中提取 `magnet` / `ed2k` 链接，按页面标题自动清洗并创建 115 网盘子目录，然后将离线任务推送到该目录中。脚本还支持可选的自动解压并删除功能，可在压缩包下载完成后解压到当前标题目录并删除原压缩包。

> 本项目仅用于用户有权下载、备份、归档和处理的合法资源。请遵守所在地区法律法规、网站规则和 115 网盘服务条款。

## 1. 功能特性
- 支持 115 Cookie 登录 / 复制 Cookie
- 支持可选 Bot 下载接口
- 自动识别帖子中的 `magnet` 和 `ed2k` 链接
- 从帖子标题中提取目录名
- 支持标题清洗：
  - 删除默认词：`【ED2K】`、`【自转】`、`【自整理】`、`压缩包`
  - 删除 `数字 + 配额`，例如 `1配额`、`12配额`
  - 删除 emoji
  - 删除 `/` 等不适合作为目录名的符号
  - 保留中文 `【】`，例如 `【1.08G/52V/1配额】` 会清洗为 `【1.08G52V】`
- 在用户设置的 115 父目录下自动创建“清洗后标题”子目录
- 将离线任务推送到：
  
  ```text
  115 父目录 / 清洗后标题 / 文件
  ```
- 支持推送后自动解压压缩包
- 支持解压后自动删除原压缩包，默认关闭
- 提供设置面板：
  - 115 父目录 ID
  - Bot 下载地址
  - 标题删除词
  - 标题最大长度
  - 是否按标题创建子目录
  - 是否推送后自动解压
  - 是否解压后删除原压缩包
  - 自动解压轮询参数
  - 默认解压密码

## 3. 安装说明
### 3.1：从 GreasyFork 安装
1. 安装 Violentmonkey 浏览器扩展（脚本依赖于油猴的GM_cookie功能，但是该功能在稳定版的油猴还没有开放，所以需要使用Beta版本来运行该脚本）。
2. 打开 `chrome://extensions/` 页面，启用开发者模式，找到 Violentmonkey 插件，点击“详情”，在“扩展选项”中找到“允许运行用户脚本”选项并启用它。
3. 访问 [98堂-115离线助手](https://greasyfork.org/zh-CN/scripts/xxxx) 页面，点击“安装此脚本”按钮。**还未发布到 GreasyFork，敬请期待**。
4. 在弹出的安装界面中确认安装（确保安装到 Violentmonkey中）。

### 3.2：从 GitHub 手动安装
1. 安装 Violentmonkey 浏览器扩展并启动“允许运行用户脚本”（同上）。
2. 打开下面的 .user.js 文件地址：
   - [98堂-115离线助手.user.js](https://raw.githubusercontent.com/Mr-jello/tang-to-115-assistant/main/tang-to-115-assistant.user.js)
3. 浏览器会提示安装用户脚本，确认安装（确保安装到 Violentmonkey中）。

## 4. 使用指南
1. 首次使用时，需要先设置 115 父目录 ID。
   - 登录 115 网盘网页版，进入你想要存储离线任务的目录。
   - 查看浏览器地址栏 URL，`https://115.com/storage/netdisk?cid=23465204XXXXXXXX&mode=wangpan`，其中 `23465204XXXXXXXX` 就是目录 ID。
   - 之后脚本会在这个目录下创建标题目录：
     ```text
     115 父目录 / 清洗后标题 / 文件
     ```
2. 标题清洗示例
   - 原标题：`【自转】【ED2K】示例资源标题【1.08G/52V/1配额】`
   - 清洗后标题：`示例资源标题【1.08G52V】`
   - 你也可以在设置面板中添加自定义删除词，一行一个。
3. 自动解压说明
   - 开启“推送后自动解压”后，脚本会在离线任务提交成功后开始监控当前标题目录。
   - 默认策略：
   ```Text
    推送成功后 10 秒开始检查
    最多启动 3 轮检查
    每轮检查 3 次
    每次间隔 20 秒
   ```
    - 如果在检查过程中发现有压缩包文件（例如 `.zip`、`.rar`、`tar`、`gz`），会自动调用 115 网盘的解压接口进行解压。
    - 可以在设置面板开启：“解压后自动删除原压缩包”，这样在解压完成后会自动删除原压缩包文件，节省空间。 该功能默认关闭。开启后，只有当解压进度达到 100% 后，脚本才会尝试删除原压缩包。

## 5. 注意事项
1. 自动解压依赖 115 网页端内部接口，接口可能随 115 更新而变化。
2. 自动解压需要页面保持打开，关闭页面后 Tampermonkey 脚本不会继续后台运行。
3. 自动删除原压缩包默认关闭，避免误删。
4. 如果压缩包需要密码，可以在设置面板填写默认解压密码。
5. 如果资源离线时间较长，可以适当调大自动解压检查次数和间隔。
6. 请仅用于合法资源的下载和备份，遵守相关法律法规和网站规则。
